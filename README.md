# AI Resume Analyzer

Frontend web app to upload resumes and receive automated analysis and improvement suggestions (PDF/DOCX). Designed as a portfolio piece demonstrating modern frontend UI and client-side processing.

---

## Features

- Upload PDF and DOCX resumes
- ATS-style scoring and keyword matching
- Skills, education, experience and project analysis
- Downloadable PDF analysis report
- Responsive, modern UI

---

## Technologies used

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## Quick start

1. Clone:

```bash
git clone https://github.com/DEEPAKS110/resume-analyser.git
cd resume-analyser
```

2. Open `index.html` in a browser (static frontend). No build required.
